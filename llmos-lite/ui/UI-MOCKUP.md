## Key UI Improvements

### Simplified Navigation
1. **Single volume selector** - Clear indication of active volume
2. **Session type badge** - Visual distinction between user/team sessions
3. **Status indicators** - Temporal vs saved, committed vs uncommitted

### Artifact Management
1. **Unified artifact view** - All artifact types in one gallery
2. **Quick actions** - View, save, reference, fork directly from cards
3. **Status badges** - Clear indication of artifact state

### Chat Enhancements
1. **@ autocomplete** - Easy artifact referencing
2. **Inline previews** - See artifacts without leaving chat
3. **Quick save** - Save artifacts directly from chat

### Context Awareness
1. **Session info panel** - Always visible session state
2. **Referenced artifacts** - Track what's being used
3. **Volume browser** - Quick access to all volumes

### Mobile Optimization
1. **Bottom navigation** - Easy thumb access
2. **Collapsible panels** - More screen space for chat
3. **Touch-optimized buttons** - Larger tap targets
